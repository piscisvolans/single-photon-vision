Zaber Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Zaber Technologies
Supported Language: LabVIEW
Supported Models: T-LSR, T-LS, T-LA, T-CD, T-JOY, T-LLS, T-MM, T-NA, T-NM, T-RS (T Series)
Models Tested: T-LSR, T-LS, T-LA, T-CD
Interface: Serial
Firmware Revisions Tested: 5.11, 5.12

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: March 20, 2009
Current Revision Date: March 20, 2009

For more documentation and a tutorial, go to the Zaber Technologies web site.

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 3.0 or later

For more details, refer to the LabVIEW Help under Help » Search the LabVIEW Help.
